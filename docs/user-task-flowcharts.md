# 心迹档案用户任务流程图（前端 2.0 预览版）

## 1. 登录进入工作台

```mermaid
flowchart TD
    A["进入首页"] --> B["查看品牌介绍与聊天演示"]
    B --> C["点击 登录进入工作台"]
    C --> D{"当前是否为预览态"}
    D -- "是" --> E["自动进入工作台"]
    D -- "否" --> F["打开手机号验证码登录弹窗"]
    F --> G["输入手机号"]
    G --> H["点击 获取验证码"]
    H --> I{"验证码发送成功?"}
    I -- "否" --> J["提示发送失败/手机号格式错误"]
    J --> G
    I -- "是" --> K["输入验证码"]
    K --> L["提交 登录/注册"]
    L --> M{"登录成功?"}
    M -- "否" --> N["提示验证码错误/登录失败"]
    N --> K
    M -- "是" --> O["写入登录态和 token"]
    O --> P["拉取用户信息/个人资料/智能体/记忆"]
    P --> E
    E --> Q["进入工作台默认模块"]
```

## 2. 创建智能体

```mermaid
flowchart TD
    A["进入 我的智能体"] --> B["看到 3 个入口"]
    B --> C{"选择哪种方式"}
    C -- "官方人设" --> D["进入官方人设编辑态"]
    C -- "创建人设" --> E["进入空白编辑态"]
    C -- "导入人设" --> F["进入导入编辑态"]
    D --> G["选择预设人格"]
    G --> H["系统自动填充字段"]
    E --> I["用户手动填写字段"]
    F --> J["系统填入导入态默认值"]
    H --> K["用户继续修改字段"]
    I --> K
    J --> K
    K --> L["点击 保存到我的智能体"]
    L --> M["先保存到本地"]
    M --> N{"是否已登录并接通后端?"}
    N -- "否" --> O["直接显示本地保存结果"]
    N -- "是" --> P["同步创建智能体到后端"]
    P --> Q{"后端保存成功?"}
    Q -- "否" --> R["提示已先保存到本地，后端稍后再试"]
    Q -- "是" --> S["更新智能体 ID 和列表"]
    O --> T["智能体出现在 我的智能体 列表"]
    R --> T
    S --> T
```

## 4. 导入人设

```mermaid
flowchart TD
    A["进入 我的智能体"] --> B["点击 导入人设"]
    B --> C["打开导入态编辑面板"]
    C --> D["系统先填入导入态默认值"]
    D --> E["名称默认显示 导入人设"]
    E --> F["用户粘贴或补充外部人设内容"]

    G --> H["右侧试聊窗口同步更新名称"]
    H --> I["点击 保存到我的智能体"]

    J --> K{"是否需要同步后端"}
    K -- "否" --> L["完成保存"]

    N -- "是" --> P["更新远端记录"]
    L --> Q["可从 我的智能体 列表再次进入"]
    O --> Q
    P --> Q
```

## 5. 智能体试聊

```mermaid
flowchart TD
    A["进入智能体详情页"] --> B["查看右侧 实时模拟对话"]
    B --> C["输入消息"]
    C --> D{"消息是否为空"}
    D -- "是" --> E["不发送"]
    D -- "否" --> F["把用户消息插入聊天区"]
    F --> G["点击发送"]
    G --> H{"是否已登录且有后端接口"}
    H -- "否" --> I["生成本地 mock 回复"]
    H -- "是" --> J["调用 /chat/simulate"]
    J --> K{"接口返回成功?"}
    K -- "否" --> L["显示聊天失败提示"]
    K -- "是" --> M["插入 AI 回复"]
    I --> N["显示 AI mock 回复"]
    L --> O["等待用户继续输入"]
    M --> O
    N --> O
```

## 6. 记忆管理

```mermaid
flowchart TD

    K -- "3D 日期点" --> N["切换到对应日期"]

    M --> O
    N --> I
```

## 7. 充值中心

```mermaid
flowchart TD
    A["进入 充值中心"] --> B["看到余额、使用占比、充值包、公告"]
    B --> C{"选择充值方式"}
    C -- "预设充值包" --> D["点击 10/25/50/100/150/200 档位"]
    C -- "自定义数量" --> E["输入自定义 token 数量"]
    D --> F["记录 selectedToken 和 selectedPrice"]
    E --> G["清空预设高亮并计算价格"]
    F --> H["点击 确认充值"]
    G --> H
    H --> I{"数量是否合法"}
    I -- "否" --> J["不更新余额/等待用户修正"]
    I -- "是" --> K["将 token 加到余额"]
    K --> L["刷新顶部余额显示"]
    L --> M["写入本地存储"]
```
